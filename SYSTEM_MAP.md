# 🧠 SYSTEM_MAP.md: 体系的知識マトリックス

| 領域 | 基礎 (Level 1) | 応用 (Level 2) | 最新トレンド (Level 3) |
| :--- | :--- | :--- | :--- |
| **将棋 (SHOGI)** | 駒の動き・定跡の型 | 特定戦法の弱点分析 | AI流のバランス重視・エルモ囲い急戦 |
| **音声 (VOICE)** | 音声認識(STT)・合成(TTS) | ノイズキャンセリング・VAD | DeepFilterNet3・FastRTC・感情認識 |

| **開発 (DEV)** | Python/JS/HTML 基礎 | パフォーマンス最適化・リファクタリング | AIエージェント自律進化ループ |
